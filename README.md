# File Manager

A modern cloud-based file management platform built with Laravel, Vue.js, TypeScript and AWS S3.

Inspired by services such as Dropbox and Google Drive, this project focuses on scalable file storage, file sharing and modern full-stack application architecture.

> **Status:** Under active development
> Demo: https://file-manager-main-hglekz.laravel.cloud/

---

## Features

- Cloud-based file storage using AWS S3
- Modern Laravel backend architecture
- Vue.js and TypeScript frontend
- Secure authentication and user management
- Upload, organize and manage files
- Share files between users
- Responsive user interface
- Docker-based development environment
- Scalable architecture designed for future expansion

## Project Goals

The goal of this project is to explore the architecture behind modern cloud storage platforms and gain hands-on experience with:

- Large-scale file management systems
- Cloud storage integrations
- Full-stack application architecture
- Authentication and authorization
- Scalable software design
- Modern development workflows

---

## Why This Project?

Most portfolio projects demonstrate isolated functionality.

This project aims to replicate real-world software challenges found in modern SaaS applications, including cloud storage, file organization, user management and scalable system design.

Rather than focusing on a single feature, the goal is to build and maintain a complete platform that evolves over time.

---

## Local Development

```bash
git clone https://github.com/jellestekelenburg/file-manager.git

cd file-manager

docker compose up -d

composer install

npm install

php artisan migrate

npm run dev
```
